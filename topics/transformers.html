1. Word Embeddings Word2Vec

One-Hot: כל מילה = וקטור באורך $|V|$ עם 1 יחיד. חסרונות: אין דמיון בין מילים (וקטורים אורתוגונליים), קללת המימד, דלילות.

Word2Vec (Skip-gram)
ההנחה: משמעות מילה נגזרת מהמילים שסביבה. מנבאים מילות הקשר בהינתן מילת מרכז: $$P(o|c)=\frac{\exp(u_o^Tv_c)}{\sum_{w\in V}\exp(u_w^Tv_c)},\qquad J(\theta)=-\tfrac1T\sum_t\sum_{-m\le j\le m,j\neq0}\log P(w_{t+j}|w_t).$$
מרחב סמנטי
מילים בהקשרים דומים מקבלות וקטורים קרובים במרחב — אפשר למדוד דמיון. בסיום, שומרים רק את וקטורי המרכז $v_w$. אימון עם SGD.

2. Tokenization BPE

הופכים טקסט לוקטורים. פיצול לפי מילים שלמות נכשל: מילים לא-מוכרות (UNK), אין קשר בין run/running, אוצר מילים ענק. הפתרון: Sub-word tokenization (BPE) — מפרקים מילים לחלקי-מילים נפוצים.

BPE
מתחילים מתווים בודדים, ובאופן איטרטיבי מאחדים את זוג התווים הנפוץ ביותר. מילים נפוצות (the) נשארות שלמות; נדירות (unhappy) מתפרקות ל-un+happy. יתרון: אין UNK, אוצר מילים מצטמצם.

3. Self-Attention לב הטרנספורמר

כל טוקן מוטל לשלושה וקטורים: Query ($q$ — מה אני מחפש), Key ($k$ — מה אני מציע), Value ($v$ — התוכן). $Q=XW^Q$, $K=XW^K$, $V=XW^V$.

נוסחת ה-Attention
$$\mathrm{Attention}(Q,K,V)=\mathrm{softmax}\!\Big(\frac{QK^T}{\sqrt{d_k}}\Big)V.$$ $QK^T$ = מטריצת התאמות; $\sqrt{d_k}$ = מנרמל ערכים גדולים; softmax = הופך להסתברויות; כפל ב-$V$ = ייצוג הקשרי חדש.
נתון: טוקן עם $q=(1,0)$, שני טוקנים עם $k_1=(1,0),\ k_2=(0,1)$, ערכים $v_1=(2,0),\ v_2=(0,4)$, $d_k=2$.
ציונים: $q\cdot k_1=1$, $q\cdot k_2=0$. מנורמל: $\tfrac1{\sqrt2}=0.707$ ⟹ $(0.707, 0)$.
Softmax: $\tfrac{e^{0.707}}{e^{0.707}+e^0}=\tfrac{2.028}{3.028}=0.67$; השני $0.33$.
פלט: $0.67\cdot v_1 + 0.33\cdot v_2 = 0.67(2,0)+0.33(0,4)=(1.34,\ 1.32)$ — שילוב משוקלל שמדגיש את הטוקן הדומה ל-Query.
Multi-Head
במקום ראש אחד, $h$ ראשים במקביל, כל אחד עם $W^Q,W^K,W^V$ משלו — לומדים סוגי קשרים שונים (תחבירי, סמנטי). הפלטים משורשרים.
Positional Encoding
Self-Attention אדיש לסדר (אינווריאנטי לפרמוטציות) — Dog bites man = Man bites dog! מזריקים מידע מיקום (סינוס/קוסינוס) ל-Embeddings לפני ה-Encoder.
Add & Norm ו-FFN
Add (Residual): $\mathrm{Output}=\mathrm{LayerNorm}(x+\mathrm{Sublayer}(x))$ — מונע דעיכת גרדיאנטים. Feed-Forward: $\mathrm{FFN}(x)=\max(0,xW_1+b_1)W_2+b_2$ — מוסיף אי-לינאריות לכל טוקן. עלות ה-Attention ריבועית באורך הרצף ($n\times n$).
